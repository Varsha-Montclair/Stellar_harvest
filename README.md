# Stellar Harvest - Space Mining Adventure

A Python Pygame game where you pilot a mining ship through asteroid fields, collecting glowing ore crystals.

## How to Play

### Requirements:
- Python 3.8 or higher
- Pygame library

### Installation:
1. Install Python from [python.org](https://python.org)
2. Install Pygame: `pip install pygame`

### To Run:
1. Download `Midterm_game.py`
2. Open terminal/command prompt in the download folder
3. Run: `python Midterm_game.py`

### Controls:
- **WASD or Arrow Keys**: Move ship
- **ESC**: Quit game
- **ENTER/SPACE**: Start game

### Gameplay:
- Collect colored ores for points (10-30 points each)
- Avoid asteroids (3 hits and game over)
- Score increases, game gets harder
